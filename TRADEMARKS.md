# Trademark And Naming Policy

This project name and related branding identify the original upstream project maintained by Jason Grey.

## You May

- Fork the repository to propose fixes and enhancements.
- Refer to the project name in factual statements, such as compatibility notes.

## You May Not

- Use the project name, logos, or branding in a way that implies your fork is the official upstream.
- Rebrand your fork as the official project or remove upstream attribution.
- Use confusingly similar names that can mislead users about origin or maintenance.

## Fork Naming Guidance

Use a clearly distinct name for public forks, and include a prominent note that your fork is unofficial.

## Attribution

Forks and redistributions must preserve copyright and license notices.

## Questions

Open an issue if you want permission for branding uses beyond this policy.
