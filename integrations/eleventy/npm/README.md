# @jt55401/eddie-eleventy

Install helper for integrating Eddie into eleventy sites.

## Usage

```bash
npx @jt55401/eddie-eleventy /path/to/site /path/to/eddie/dist
```

Arguments:

- First argument: CMS site root directory
- Second argument: Eddie runtime asset directory containing:
  - `eddie-widget.js`
  - `eddie-worker.js`
  - `eddie-wasm.js`
  - `eddie.wasm`
